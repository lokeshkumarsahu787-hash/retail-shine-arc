# 🛍️ Retail Shine Arc

A modern and responsive E-Commerce Web Application built with React, TypeScript, Vite, and Tailwind CSS.
This project delivers a premium shopping experience with clean UI/UX, dynamic product browsing, cart management, filtering, and modern frontend architecture.

---

## 🚀 Features

* 🔍 Product Search & Filtering
* 🛒 Add to Cart Functionality
* 📦 Product Detail Pages
* 🧾 Dynamic Cart Management
* 🎨 Modern Responsive UI
* ⚡ Fast Performance with Vite
* 🌙 Beautiful UI Components
* 📱 Mobile-Friendly Design
* 🧠 Clean Component Architecture
* 🛡️ TypeScript Support

---

## 🛠️ Tech Stack

### Frontend

* React.js
* TypeScript
* Vite
* Tailwind CSS
* React Router DOM

### UI & Styling

* shadcn/ui

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/sourabh0907/retail-shine-arc.git
```

Navigate into the project folder:

```bash
cd retail-shine-arc
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
